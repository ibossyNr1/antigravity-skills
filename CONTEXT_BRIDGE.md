## What is a Context Package?

A `CLIENT_CONTEXT.md` file containing your company's:

- **Core Identity** — mission, values, positioning, founding context
- **Brand Voice** — tone, messaging pillars, communication guidelines
- **Target Audience** — ICPs, pain points, motivations, preferred channels
- **Product/Service** — offerings, pricing, integrations, tech stack
- **Market Landscape** — competitors, differentiation, SWOT analysis
- **Visual Guidelines** — colors, typography, design language
