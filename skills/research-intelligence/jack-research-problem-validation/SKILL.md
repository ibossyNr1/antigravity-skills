## When to Use

Use this skill when you need to:

## What This Does

Use GummySearch to monitor Reddit communities for keywords like "I hate it when..." or "Is there an alternative to..." to identify user pain points.

## Workflow

1. Identify relevant Reddit communities (e.g., r/realestate, r/shopify).
2. Set up GummySearch to track these communities.
3. Monitor keywords such as "I hate it when..." or "Is there an alternative to...".
4. Analyze the search results to identify common frustrations and potential SaaS ideas.
