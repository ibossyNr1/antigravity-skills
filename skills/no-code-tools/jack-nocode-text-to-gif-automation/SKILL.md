# Nocode Text To Gif Automation

## When to Use

Use this skill when you need to:
- when needing to automatically convert text input into animated GIFs
- when creating content from user-submitted text

## What This Does

Blueprint for a text-to-GIF automation using Tally, Make, Replicate, Luma Labs, Carrd, and Giphy.

## Workflow

1. Capture text input via Tally form.
2. Use Make.com to orchestrate the workflow.
3. Generate image from text using Replicate or similar.
4. Animate the image using Luma Labs.
5. Host the GIF on Giphy or similar platform.
6. Display/sell via Carrd landing page.

## Configuration

**Required tools/platforms:**
- tally
- make.com
- replicate
- luma labs
- carrd
- giphy
