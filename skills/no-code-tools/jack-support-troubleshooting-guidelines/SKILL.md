# Support Troubleshooting Guidelines

## When to Use

Use this skill when you need to:
- when encountering issues with AI agents in Make.com
- when asking for support with AI automation setups

## What This Does

Provides a structured approach for troubleshooting AI agent issues, including detailing the problem, steps tried, and error messages.

## Workflow

1. Be specific about your agent setup issue - Tell us exactly which part of the process you're struggling with (agent creation, tool scenarios, inputs/outputs, triggering, etc.)
2. Show us what's happening - (Ideally) record a 2-3 minute Loom video demonstrating the problem or share screenshots of your agent configuration and tool scenarios with the exact errors you're seeing
3. Structure your question efficiently:
    - Start with a one-sentence summary of your issue
    - List the exact steps you've already tried
    - Include error messages (copy/paste the exact text)
