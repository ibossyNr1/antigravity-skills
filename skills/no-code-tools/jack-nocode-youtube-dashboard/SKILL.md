# Nocode Youtube Dashboard

## When to Use

Use this skill when you need to:
- when creating a YouTube analytics dashboard without coding
- when needing a quick and easy way to visualize YouTube data

## What This Does

Build a YouTube Growth Dashboard using Loveable + N8N, a beginner-friendly approach for no-code/low-code development.

## Workflow

1. Set up Loveable and connect it to your YouTube account.
2. Use N8N to automate data extraction from YouTube analytics.
3. Design the dashboard in Loveable to display key metrics.
4. Configure N8N to update the dashboard with the latest data.
