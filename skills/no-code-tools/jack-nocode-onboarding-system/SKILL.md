# Nocode Onboarding System

## When to Use

Use this skill when you need to:
- when building an automated customer onboarding system
- when needing to rapidly prototype a user dashboard

## What This Does

Use GoHighLevel, Claude, Google AI Studio, Supabase, and Vercel to build a complete automated onboarding system.

## Workflow

1. **GoHighLevel**: Use for CRM, onboarding emails, and webhook automations.
2. **Claude**: Employ Claude for SOP creation, planning, coding, and admin logic.
3. **Google AI Studio**: Use to generate front-end dashboards and rapidly iterate on the UI.
4. **Supabase**: Set up Supabase for database, authentication, tables, and row-level security.
5. **Vercel**: Deploy the application live with one-click deployment and custom domains.
