# Content Ideation Prompt

## When to Use

Use this skill when you need to:
- when brainstorming SaaS ideas
- when identifying market opportunities

## What This Does

Use these prompts to generate SaaS ideas based on existing tools' shortcomings.

## Workflow

- "[Competitor Name] + sucks"
- "How do I + [Task] + without + [Expensive Tool]"
- "Is there a simple tool for..."
- "I would pay for an app that..."
