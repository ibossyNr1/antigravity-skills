## When to Use

## What This Does

Convert video files to WebP format for smaller file sizes and better web performance using EZgif.

## Workflow

1. Go to EZgif: https://ezgif.com/video-to-webp
2. Upload your video file.
3. Adjust settings as needed (e.g., resolution, frame rate).
4. Convert the video to WebP format.
5. Download the optimized WebP file.
