# Video Luma Image To Video Prompt

## When to Use

Use this skill when you need to:
- when converting static images into animated videos
- when needing to add dynamic backgrounds to a static subject

## What This Does

Generates a prompt for Luma Labs to create a video from an image with animated background.

## Workflow

A person standing still and centered in the foreground, remaining completely motionless, while the background smoothly animates with dynamic and varying movements. The animation should focus on continuous, flowing changes in the background, allowing the environment to vary each time, while the person stays entirely static
