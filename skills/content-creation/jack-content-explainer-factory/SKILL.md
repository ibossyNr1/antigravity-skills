# Content Explainer Factory

## When to Use

Use this skill when you need to:
- when explaining complex topics
- to create easy-to-understand content

## What This Does

Procedure to turn complex topics into simple content using AI to generate explainer documents, videos and flashcards.

## Workflow

Complex Topic (e.g., "Quantum Computing")
        ↓
Research: Fundamentals + latest developments
        ↓
Generate:
  → ELI5 briefing doc
  → Kawaii-style explainer video
  → Visual infographic
  → Flashcards for key terms
        ↓
Build: Interactive learning module
