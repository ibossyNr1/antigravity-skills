# Design Find Ui Inspiration

## When to Use

Use this skill when you need to:
- When starting a new web design project
- When seeking inspiration for UI/UX elements

## What This Does

Use curated website galleries like Awwwards, CSS Design Awards, Behance and Dribbble to find inspiration for UI/UX and web design projects.

## Workflow

1. Visit a UI/UX design inspiration website (e.g., Awwwards, CSS Design Awards, Behance, Dribbble).
2. Browse the curated collections of websites and UI designs.
3. Note design elements, layouts, and interactions that resonate with your project.
4. Use these inspirations as a foundation for your own designs.
