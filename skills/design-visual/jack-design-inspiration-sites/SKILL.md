# Design Inspiration Sites

## When to Use

Use this skill when you need to:
- when seeking inspiration for web design projects
- when looking for UI/UX design patterns

## What This Does

Provides a list of websites for web/UI design inspiration, covering general design, component-level inspiration, trends, UX, and app-focused designs.
