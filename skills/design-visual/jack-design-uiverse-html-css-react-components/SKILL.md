# Design Uiverse Html Css React Components

## When to Use

Use this skill when you need to:
- When needing a specific UI element
- When seeking open-source UI components

## What This Does

Leverage Uiverse.io to find open-source UI components for web projects, available as HTML/CSS, Tailwind, React snippets, and Figma-compatible code.

## Workflow

1. Visit Uiverse.io.
2. Search for UI elements like buttons, forms, loaders, etc.
3. Filter by technology (HTML/CSS, Tailwind, React).
4. Copy the code snippet and integrate it into your project.
5. Customize the component as needed.
