# Design Ui Inspiration List

## When to Use

Use this skill when you need to:
- when designing a website
- when seeking UI/UX inspiration

## What This Does

A curated list of websites to find UI/UX inspiration for web design.

## Workflow

- Awwwards – Curated award-winning websites with great UX & creativity. (https://www.awwwards.com)
- CSS Design Awards – Daily picks of innovative, high-quality web designs. (https://www.cssdesignawards.com)
- Behance – Huge library of UI/UX and digital design projects. (https://www.behance.net)
- Dribbble – Designers share shots of UI elements, components & screens. (https://dribbble.com)
- Siteinspire – Clean presentation of curated web designs by style and type. (https://www.siteinspire.com)
- Land-book – Massive collection of great landing page examples. (https://land-book.com)
- Collect UI – Daily UI inspiration based on real screens. (https://collectui.com)
- UI Movement – Handpicked UI design inspiration sorted by trends. (https://uimovement.com)
- UI Garage – UI patterns & examples for apps & websites. (https://uigarage.net)
- Mobbin – Curated mobile UX patterns from top apps. (https://mobbin.com)
- Pttrns – UI patterns for mobile screens and interactions. (https://pttrns.com)
- UI Patterns – Common UI/UX patterns with examples. (https://ui-patterns.com)
- Design Systems Repo – Examples of design system components. (https://designsystemsrepo.com)
- Little Big Details – Focus on subtle interaction and micro-UX. (http://littlebigdetails.com)
- One Page Love – Best one-page websites. (https://onepagelove.com)
- Httpster – Indie & creative web design showcase. (https://httpster.net)
- Bestfolios – Portfolio inspiration and real case studies. (https://bestfolios.com)
- Niice – Collaborative moodboards & visual inspiration library. (https://niice.co)
- Designspiration – Pinterest-style visual ideas. (https://www.designspiration.com)
- Pinterest – Search UI design trends, UX layouts for tons of boards.
