# mcp2cli — One CLI for Every API

## When to Use

Use when you want to interact with MCP servers via CLI instead of through an LLM, or when you need to reduce token usage by 96-99% compared to native MCP tool calls. Useful for scripting, automation, and batch operations against MCP-compatible APIs.

## Description

mcp2cli converts any MCP server into a traditional CLI tool. Instead of routing every API call through an LLM (which consumes tokens), it generates direct CLI commands. 144 HN points and 103 comments indicate high developer interest.
