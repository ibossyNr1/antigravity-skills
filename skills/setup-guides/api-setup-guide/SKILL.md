# Api Setup Guide

## When to Use

When integrating Clearbit, Beehiiv, Calendly, Slack, or Twitter APIs into a project.

## What This Does

Sets up API keys and webhooks for Clearbit, Beehiiv, Calendly, Slack, and Twitter. Covers account creation, key retrieval, and testing.

## Workflow

1. Set up Clearbit API
2. Set up Beehiiv API
3. Set up Calendly Link
4. Set up Slack Webhook URL
5. Set up Twitter/X API

## Prerequisites

Valid email addresses, credit card (for some services), and access to your website/domain.

## Verification

Verify each service using provided curl commands and browser checks to ensure proper setup and connectivity.
