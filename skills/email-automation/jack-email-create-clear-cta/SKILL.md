# Email Create Clear Cta

## When to Use

Use this skill when you need to:
- When designing email layouts
- To improve click-through rates and sales

## What This Does

Use single, prominent CTAs in each email. Focused approach increases clicks and sales. Make buttons stand out with action text.

## Workflow

1. Focus on a single, primary call-to-action (CTA) for each email.
2. Make the CTA prominent and easily noticeable.
3. Use action-oriented text on CTA buttons.
4. Use contrasting colors to make the CTA stand out.
